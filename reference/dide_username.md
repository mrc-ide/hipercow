# Report DIDE username

Report the username used to log into the web portal for use with the
DIDE cluster. This may or may not be the same as your local username. We
may ask you to run this when helping debug cluster failures.

## Usage

``` r
dide_username()
```

## Value

Your username, as a string

## Examples

``` r
if (FALSE) {

# Return your DIDE username
dide_username()
}
```
