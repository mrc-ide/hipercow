# The default set of environment variables used for all tasks.

These are chosen to provide the best general experience. The default
variables may be overridden globally by setting the
hipercow.default_envvars option, or on a per-task and variable basis by
assigning a different value to the environment variable in question.

## Usage

``` r
DEFAULT_ENVVARS
```

## Format

An object of class `hipercow_envvars` (inherits from `data.frame`) with
1 rows and 3 columns.
